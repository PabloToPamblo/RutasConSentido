# 🌍 RutasConSentido – Code With a Purpose

**RutasConSentido** is not just a web platform.  
It’s a movement — built with love, logic, and the belief that **technology can change lives**.

This project is inspired by my journey documenting stories from people in need around the world through my [YouTube channel](https://www.youtube.com/@RutasConSentido). Every line of code written here is meant to **create real, measurable, positive impact** — not in theory, but in action.

---

## 🎯 What Are We Building?

We’re building a platform where users can:

- 🔐 Log in with their **Google account**
- ❤️ Earn points by **liking videos**, **sharing content**, and **inviting others**
- 🎁 Unlock **achievements** and **exclusive content**
- 🌱 Most importantly: **help people in need**, just by participating

We don’t ask for donations.  
We simply invite people to take part in a system where **likes**, **shares**, and **community participation** turn into real-world help.

> This is code with a conscience.  
> This is tech for good.  
> This is **RutasConSentido**.

---

## 🧱 Tech Stack

We’re using a full modern DevSecOps-ready stack:

- **Backend**: Django + Django REST Framework  
- **Frontend**: React + TailwindCSS  
- **Auth**: Google OAuth2  
- **Database**: PostgreSQL  
- **CI/CD**: GitHub Actions + Jenkins  
- **Dockerized** and deployed with **Kubernetes**  
- **Secure by design** with `.env`, Bandit, Trivy  
- **Test-Driven Development (TDD)** and **Behavior-Driven Development (BDD)** from day one
